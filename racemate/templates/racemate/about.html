Racemate is a running application for:

  • Calculate VDOT(efficiency) and training tempos in accordance with the assumptions of the running trainer Jack Daniels
  • Predict finish times at common race distances
  • Communication between members of the running group *
  • Creating a database of completed trainings *
  • Monitoring and updating the user's form *
  • Creating personalized running plans *

    • * features available only for logged user

    Technologies:
    Python, Django, PostgreSQL, JavaScript
    IDE: PyCharm
    CVS:: Git
    In this project I used SB Admin template
